Gangaji

A beautiful, interactive profiler for Bazel builds. Visualize your build performance with flamegraphs and get actionable insights.

Installation

Get up and running in one command.

Shell 
aspect axl add gh:thesayyn/gangaji

Usage

Profile your Bazel builds.

Shell 
aspect profile //targets

Flamegraph View

Interactive flamegraph visualization to explore build actions hierarchically.

Search & Filter

Quickly find specific actions and filter by target, duration, or type.

Smart Suggestions

Get actionable recommendations to improve your build performance.
